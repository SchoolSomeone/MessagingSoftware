# Security Policy

## Supported Versions

Currently, versions beyond 0.3.5 will be supported.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :x:                |
| 0.2.x   | :x:                |
| 3.0.0   | :x:                |
| < 0.3.5 | :white_check_mark: |

## Reporting a Vulnerability

Currently, security vulnerabilities cannot be reported.
